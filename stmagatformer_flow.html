PHASE 06Training Strategy
🔄 LOSO (Primary)
Leave-One-Subject-Out
Train on N-1 subjects
Test on held-out subject
→ fold_###.json per fold
📊 Within-Subject (Secondary)
70% train / 15% val / 15% test
Early stopping on val F1
Patience = 20 epochs
⚙️ Regularization
Dropout 0.3–0.5
Label smoothing 0.1
Gradient clipping (GRU)
AdamW weight decay
Multi-Task Loss
ℒ = α·ℒ_aff + β·ℒ_cog + γ·ℒ_clin + λ·ℒ_reg  — activated only if labels exist for the dataset
↓

Score: P_clin ∈ [0,1]

PHASE 08Window → Subject Aggregation
Each subject has N overlapping windows, each producing a probability vector
p̄ˢ = (1/N) Σᵢ pᵢˢ → mean across all windows = subject-level probabilities [P_aff, P_cog, P_clin]
Also compute: Var(pᵢˢ) = variance across windows → triggers Inconclusive if > threshold τ
↓
PHASE 09Subject-Level Decision Logic — Final Label
P_aff > 0.6
AND P_cog ≤ 0.6
😤 AFFECTIVE
P_cog > 0.6
AND P_aff ≤ 0.6
🧩 COGNITIVE
P_aff > 0.6
AND P_cog > 0.6
🔀 AFFECTO-COGNITIVE
P_clin > 0.7
(any above)
🏥 CLINICALLY IMPAIRED
Var(pᵢˢ)
> threshold τ
❓ INCONCLUSIVE
